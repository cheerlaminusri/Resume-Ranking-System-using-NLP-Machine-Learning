# Resume Ranking System (NLP + ML)
NLP-based Resume Ranking System using TF-IDF and Cosine Similarity.

## Overview
A Flask-based web application that ranks resumes based on job description similarity using TF-IDF vectorization and cosine similarity.

## Tech Stack
- Python
- Flask
- Scikit-learn
- NLP (TF-IDF)
- Cosine Similarity

## Features
- Upload multiple resumes (PDF/DOCX/TXT)
- Extract text automatically
- Rank resumes based on similarity
- Display Top 5 matching resumes

## Run Locally
pip install -r requirements.txt
python app.py
